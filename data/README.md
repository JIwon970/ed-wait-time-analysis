# Cleaned Dataset (Database)

This folder contains the cleaned Emergency Department dataset stored as a database file.

## Description
- The original CSV file was cleaned in Excel.
- The final cleaned dataset was then saved as a database table.
- This database is used for all SQL analysis in this project.

## Table Name
- ED_dataset_clean

## Columns Included
- Date
- Weekday
- Hospital_Name
- Triage_Category
- Time
- Hour
- Waiting_Time_Min
- Treatment_Time_Min
- Crowding_Level
- Peak_Hour

## Notes
- This dataset is **synthetic** and does **not contain real patient data**.
- It is used only for portfolio and learning purposes.
