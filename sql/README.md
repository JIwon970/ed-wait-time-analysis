# SQL Analysis

This folder contains SQL queries used to analyze the cleaned Emergency Department dataset.

## Database Used
- Table name: ED_dataset_clean

## Files Included

### 1. ed_kpi_queries.sql
This file contains KPI analysis queries:
- Total number of patients
- Minimum waiting time
- Maximum waiting time
- Average waiting time
- Median waiting time
- Average peak-hour waiting time

### 2. ed_group_analysis.sql
This file contains group-based analysis queries:
- Average waiting time by hospital
- Average waiting time by triage category
- Average waiting time by weekday
- Average waiting time by hour
- Peak vs non-peak hour comparison

## Purpose
These SQL queries were used to:
- Validate Excel dashboard KPI values
- Support the findings shown in the visuals and report
