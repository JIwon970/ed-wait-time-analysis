# Emergency Department Waiting Time Analysis - Project Report

This folder contains the final PDF report for the Emergency Department (ED) waiting time analysis project.

## Report Contents
- Project Overview  
- Dataset Description  
- Data Preparation  
- Exploratory Data Analysis (EDA)  
- Key Findings  
- Recommendations  
- Limitations  
- Power BI Dashboard Overview  

## Purpose
This report presents a complete end-to-end data analysis workflow using synthetic ED data.
It summarizes the analytical process and highlights clear, actionable insights for both technical and non-technical audiences.
